Media Server API Documentation

This is a simple API for managing a collection of movies, TV series, and songs.

Base URL

http://localhost:3000

Endpoints

Movies

GET /movies - Get all movies

Returns a list of all movies in the database.

Example response:

{
  "movies": [
    {
      "id": 1,
      "title": "Sicario",
      "year": 2015,
      "director": "Denis Villeneuve",
      "genre": "Crime"
    },
    ...
  ]
}

POST /movies - Add a new movie

Adds a new movie to the database.

Example request body:

{
  "title": "Inception",
  "year": 2010,
  "director": "Christopher Nolan",
  "genre": "Sci-Fi"
}

PUT /movies/:id - Update a movie

Updates an existing movie in the database.

Example request body:

{
  "title": "Updated Movie Title",
  "year": 2022
}

DELETE /movies/:id - Delete a movie

Deletes a movie from the database.

TV Series

GET /series - Get all TV series

Returns a list of all TV series in the database.

Example response:

{
  "series": [
    {
      "id": 1,
      "title": "Black Mirror",
      "seasons": 6,
      "creator": "Charlie Brooker",
      "genre": "Sci-Fi"
    },
    ...
  ]
}

POST /series - Add a new TV series

Adds a new TV series to the database.

Example request body:

{
  "title": "Breaking Bad",
  "seasons": 5,
  "creator": "Vince Gilligan",
  "genre": "Drama"
}

PUT /series/:id - Update a TV series

Updates an existing TV series in the database.

Example request body:

{
  "title": "Updated Series Title",
  "seasons": 8
}

DELETE /series/:id - Delete a TV series

Deletes a TV series from the database.

Songs

GET /songs - Get all songs

Returns a list of all songs in the database.

Example response:

{
  "songs": [
    {
      "id": 1,
      "title": "Jerusalema",
      "artist": "Master KG ft. Nomcebo Zikode",
      "year": 2019,
      "genre": "House"
    },
    ...
  ]
}

POST /songs - Add a new song

Adds a new song to the database.

Example request body:

{
  "title": "New Song",
  "artist": "Artist Name",
  "year": 2023,
  "genre": "Pop"
}

PUT /songs/:id - Update a song

Updates an existing song in the database.

Example request body:

{
  "title": "Updated Song Title",
  "year": 2022
}

DELETE /songs/:id - Delete a song

Deletes a song from the database.
